# 📸 لوکل امیجز - اردو گائیڈ

## ✅ کیا شامل کیا گیا

اب وینڈر ڈیش بورڈ میں **دو طریقے** سے تصاویر شامل کر سکتے ہیں:

### 1️⃣ **لوکل سسٹم سے تصاویر** (نیا!)
- موبائل یا کمپیوٹر سے براہ راست تصاویر منتخب کریں
- 3 تصاویر تک شامل کر سکتے ہیں
- جمع کرنے سے پہلے پری ویو دیکھیں

### 2️⃣ **Cloudinary URLs** (پہلے سے موجود)
- Cloudinary کے لنکس پیسٹ کریں
- پیشہ ورانہ تصاویر کے لیے بہترین

---

## 📱 استعمال کا طریقہ

### **آپشن 1: لوکل تصاویر استعمال کریں**

**مرحلہ 1:** Add Product کھولیں
```
"+ Add Product" بٹن پر کلک کریں
```

**مرحلہ 2:** تصاویر منتخب کریں
```
"Pick Images from Device" پر کلک کریں
↓
فائل منتخب کرنے کی ونڈو کھلے گی
↓
1 سے 3 تصاویر منتخب کریں
↓
تصاویر کا پری ویو دیکھائی دے گا
```

**مرحلہ 3:** پروڈکٹ کی تفصیلات بھریں
```
Product Name: Chicken Shawarma
Price: 1200
Stock Quantity: 50
Category: Food
```

**مرحلہ 4:** جمع کرائیں
```
"Add Product" بٹن دبائیں
↓
تصاویر اپ لوڈ ہوں گی
↓
پروڈکٹ تصاویر کے ساتھ لسٹ میں آ جائے گا
```

---

### **آپشن 2: Cloudinary URLs استعمال کریں**

**مرحلہ 1:** Cloudinary پر تصاویر اپ لوڈ کریں
```
cloudinary.com پر جائیں
تصاویر اپ لوڈ کریں
URLs کاپی کریں
```

**مرحلہ 2:** URLs فارم میں ڈالیں
```
Image URL 1: https://res.cloudinary.com/.../image1.jpg
Image URL 2: https://res.cloudinary.com/.../image2.jpg
Image URL 3: https://res.cloudinary.com/.../image3.jpg
```

**مرحلہ 3:** جمع کرائیں
```
باقی تفصیلات بھریں
"Add Product" دبائیں
پروڈکٹ تصاویر کے ساتھ ظاہر ہو گا
```

## ⚡ فوری موازنہ

| خصوصیت | لوکل تصاویر | Cloudinary URLs |
|---------|-------------|-----------------|
| **کہاں سے** | موبائل/کمپیوٹر | Cloudinary cloud |
| **اپ لوڈ** | فوری طور پر | پہلے سے اپ لوڈ |
| **رفتار** | تھوڑا سست | تیز |

## 💡 بہترین استعمال

### کب لوکل تصاویر استعمال کریں:
```
✅ موبائل سے فوٹو لیں
✅ فوری طور پر شامل کرنا ہو
✅ سادہ پروڈکٹ فوٹوز
✅ ذاتی استعمال
```

### کب Cloudinary URLs استعمال کریں:
```
✅ پیشہ ورانہ تصاویر
✅ بڑی سائز کی تصاویر
✅ پہلے سے ایڈٹ شدہ
✅ بار بار استعمال
```

## 🎯 مثالیں

### **مثال 1: موبائل سے تصاویر**
```
1. موبائل سے 3 فوٹو لیں
2. "Pick Images from Device" پر کلک
3. فوٹوز منتخب کریں
4. پری ویو دیکھیں
5. Add Product دبائیں
✅ تصاویر اپ لوڈ ہو جائیں گی
```

### **مثال 2: پیشہ ورانہ تصاویر**
```
1. Cloudinary پر اچھی تصاویر اپ لوڈ کریں
2. URLs کاپی کریں
3. فارم میں پیسٹ کریں
4. Add Product دبائیں
✅ تصاویر فوری طور پر لنک ہو جائیں گی
```

### **مثال 3: دونوں ملا کر**
```
1. 2 لوکل تصاویر منتخب کریں
2. 1 Cloudinary URL پیسٹ کریں
3. Add Product دبائیں
✅ لوکل تصاویر استعمال ہوں گی (ترجیح)
```

## 🆘 مسائل اور حل

### **مسئلہ:** تصاویر منتخب نہیں ہو رہیں؟
**حل:**
- براؤزر کی پرمیشن چیک کریں
- Chrome استعمال کریں
- Mobile پر گیلری کی پرمیشن دیں

### **مسئلہ:** اپ لوڈ میں وقت لگ رہا ہے؟
**حل:**
- چھوٹی سائز کی تصاویر استعمال کریں
- یا Cloudinary URLs استعمال کریں (تیز)

### **مسئلہ:** 3 سے زیادہ تصاویر چاہیے؟
**حل:**
- فی الوقت صرف 3 تصاویر allowed ہیں
- بہترین 3 تصاویر منتخب کریں
